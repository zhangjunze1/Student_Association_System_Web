
##  本项目是学生社团管理系统一个基于Spring Boot + Vue.js构建的一个前后端分离项目框架的后端项目代码部分。  
###  后端技术栈
+ SpringBoot   
+ Mybatis plus
+ Oss 对象存储  
+ swagger2+knife4j： http://localhost:8085/doc.html

## Project setup
```
npm install
```

npm run serve
